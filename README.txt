KRISTINA COACHING DEMO PROJEKTS

Struktūra:
- index.html                -> pilnā sākumlapa
- coaching.html             -> Par koučingu
- pakalpojumi.html          -> Pakalpojumi
- cenas.html                -> Cenas
- atsauksmes.html           -> Atsauksmes
- par-mani.html             -> Par mani
- partneriem.html           -> Sadarbības partneriem
- privatais-kabinets.html   -> Privātais kabinets

Mapes:
- assets/css/styles.css     -> kopējais stils visām lapām
- assets/js/main.js         -> mobilās izvēlnes darbība
- assets/images/            -> AI ģenerētie attēli + SVG vizuālie elementi

Svarīgi:
- Header un footer ir vienoti visām lapām.
- Dizains ir responsive un pielāgots telefonam.
- Index lapa ir uztaisīta tuvā vizuālajā stilā kā apstiprinātajā mockup.
- Papildu lapas ir jau sagatavotas ar saturisku karkasu tālākai aizpildīšanai.

Lai palaistu:
1. Atver index.html pārlūkā
2. Vai augšupielādē visu projektu kā statisku mājaslapu

Piezīme:
- Kontaktu un rezervācijas linki ir demo režīmā.
- Valodu pogas ir vizuāli sagatavotas, bet vēl nepārslēdz saturu.
- Google Calendar rezervāciju integrācija un apmaksa vēl nav pieslēgta — tas ir nākamais solis.
